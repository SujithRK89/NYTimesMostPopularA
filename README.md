# NYTimesMostPopular

# Pattern used MVVM

# Build the project from the command-line

To build a debug version of NYTimesMostPopular Android application, you can run ./gradlew assembleDebug from the root of repository. In a default project setup, the resulting apk can then be found in app/build/outputs/apk/app-debug.apk.

# Jacoco Coverage report

Run the below code from the root of repository in Terminal

./gradlew createDebugCoverageReport

and report will be generated in 
/build/reports/coverage/debug

# Run static code analysis such as linting

Run the below code from the root of repository in Terminal

./gradlew lint

# SonarQube

Run the below code from the root of repository in Terminal
./gradlew sonar
